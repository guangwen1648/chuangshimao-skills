# chuangshimao-skills

基于 GPT-Image-2 模型的 AI 图像生成技能。

**🌐 注册使用**: https://chuangshimao.com

## 功能特性

- 文生图：输入文字描述，生成精美图片
- 图生图：上传参考图，AI 帮你改图
- 15种比例：1:1、16:9、9:16 等
- 1K高清：高质量输出
- 中文支持：完美支持中文提示词
- 极速生成：约 30 秒出图

## 快速开始

1. 访问 **https://chuangshimao.com** 注册账号
2. 进入 **个人中心 → API Key 管理** 获取密钥
3. 开始生成图片！
