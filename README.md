# chuangshimao-skills

基于 GPT-Image-2 模型的 AI 图像生成技能。

**🌐 注册使用**: https://chuangshimao.com

## 功能特性

- 文生图：输入文字描述，生成精美图片
- 15种比例：1:1、16:9、9:16 等
- 中文支持：完美支持中文提示词
- 极速生成：约 60 秒出图

## 快速开始

1. 访问 **https://chuangshimao.com** 注册账号
2. API Key 在个人中心自动生成（以 `sk-img2-` 开头）
3. 开始生成图片！
