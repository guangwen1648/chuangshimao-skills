# chuangshimao-skills

基于 GPT-Image-2 模型的强大 AI 图像生成技能。

## 服务地址

**API 基础地址**: `https://api.chuangshimao.com/api/v1`

## 功能特性

- **文生图**：输入文字描述，生成精美图片
- **图生图**：上传参考图，AI 帮你改图
- **15种比例**：支持 1:1、16:9、9:16 等常用比例
- **1K高清**：输出高质量 1K 分辨率图片
- **中文支持**：完美支持中文提示词
- **极速生成**：约 30 秒出图

## 收费标准

**按量付费：¥0.1/张**

不收月费，不设套餐，用多少付多少。

## 快速开始

### 第一步：获取 API Key

1. 访问 **https://chuangshimao.com** 注册账号
2. 注册成功后，API Key 会在个人中心自动生成（以 `sk-img2-` 开头）
3. 无需手动创建，直接复制使用即可

### 第二步：配置技能

```bash
# 将 API Key 设置为环境变量
export IMG2_API_KEY="sk-img2-your-key-here"
```

### 第三步：生成你的第一张图

```json
{
  "prompt": "一只橘色的猫坐在窗台上晒太阳，宫崎骏动画风格",
  "size": "1:1",
  "resolution": "1k"
}
```

## API 文档

### 请求地址

```
POST https://api.chuangshimao.com/api/v1/generate
```

### 请求头

```
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| prompt | string | 是 | 图片描述（支持中文） |
| size | string | 否 | 图片比例，默认 1:1 |
| resolution | string | 否 | 输出分辨率，默认 1k |
| image_url | string | 否 | 参考图 URL（用于图生图） |

### 支持的图片比例

| 比例 | 说明 |
|------|------|
| 1:1 | 正方形 |
| 3:2 | 横图 |
| 2:3 | 竖图 |
| 4:3 | 标准横图 |
| 3:4 | 标准竖图 |
| 16:9 | 电影宽屏 |
| 9:16 | 手机竖屏 |
| 2:1 | 超宽横图 |
| 1:2 | 超窄竖图 |
| 3:1 | 横版Banner |
| 1:3 | 竖版Banner |
| 21:9 | 电影级超宽屏 |
| 9:21 | 超长竖图 |
| 5:4 | 经典画框 |
| 4:5 | Instagram风格 |

## 最佳实践

### 如何写好提示词

1. **具体描述**：包含主体、场景、风格
2. **指定风格**：如"数字绘画"、"摄影作品"、"宫崎骏动画"
3. **加入光线**："自然光"、"影棚灯光"、"黄金时段"
4. **添加氛围**："神秘感"、"温馨"、"戏剧性"

### 提示词示例

```
负面示例：一只猫
正面示例：一只橘色英国短毛猫，蓝眼睛，毛发蓬松，窗边自然光照射

负面示例：风景
正面示例：日出时分的海边，沙滩上有脚印，天空呈橙红色，摄影作品，4K
```

## 错误处理

| 错误码 | 错误信息 | 解决方案 |
|--------|----------|----------|
| 400 | prompt不能为空 | 请提供图片描述 |
| 401 | API Key无效或已过期 | 检查 API Key 是否正确 |
| 402 | 余额不足，请充值 | 去充值页面充值 |
| 429 | 请求过于频繁，请稍后再试 | 稍等片刻后重试 |
| 500 | 生成失败，请稍后重试 | 联系客服处理 |
